How far has the dog gone?


Description : This program calculates how much of a 3 foot hole that a dog has dug. The user must enter the current number of digs that their dog has made into a text box. Then they must press a button to find out the answer. The button will produce the percentage of the hole that the dog has already dug in a message below it.


On average, a dog usually digs a hole that is 3 feet deep.

   The formula for this calculation is :  (  CurrentDigsAmount   /   AverageDigs  )   X  100
